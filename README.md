# KTH-tic-tac-toe

## Directories

* `ui-client` <br>
Contains the code for the interaction of the game.
* `ai-client`<br>
Contains the code for the AI the player will compete against.
* `shared`<br>
Contains the code that will be shared between the clients (ie game board and protocol).
